2. 数据库表结构

t_user表

字段名 数据类型 约束 描述
id BIGINT PRIMARY KEY 用户ID
username VARCHAR(255) NOT NULL 用户名
password VARCHAR(255) NOT NULL 密码
phone VARCHAR(20) NOT NULL 手机号
role_type VARCHAR(20) NOT NULL 角色类型(user/landlord)
status VARCHAR(20) NOT NULL 状态
created_at DATETIME NOT NULL 创建时间
updated_at DATETIME NOT NULL 更新时间

t_homestay表

字段名 数据类型 约束 描述
id BIGINT PRIMARY KEY 民宿ID
landlord_id BIGINT NOT NULL, FOREIGN KEY 房东ID
name VARCHAR(255) NOT NULL 民宿名称
address VARCHAR(255) NOT NULL 地址
longitude DOUBLE NOT NULL 经度
latitude DOUBLE NOT NULL 纬度
status VARCHAR(20) NOT NULL 状态
created_at DATETIME NOT NULL 创建时间
updated_at DATETIME NOT NULL 更新时间

t_experience表

字段名 数据类型 约束 描述
id BIGINT PRIMARY KEY 体验ID
landlord_id BIGINT NOT NULL, FOREIGN KEY 房东ID
name VARCHAR(255) NOT NULL 体验名称
price DECIMAL(10,2) NOT NULL 价格
duration INT NOT NULL 时长(分钟)
longitude DOUBLE NOT NULL 经度
latitude DOUBLE NOT NULL 纬度
status VARCHAR(20) NOT NULL 状态
created_at DATETIME NOT NULL 创建时间
updated_at DATETIME NOT NULL 更新时间

t_order表

字段名 数据类型 约束 描述
id BIGINT PRIMARY KEY 订单ID
user_id BIGINT NOT NULL, FOREIGN KEY 用户ID
type VARCHAR(20) NOT NULL 类型(homestay/experience)
target_id BIGINT NOT NULL 目标ID
total_price DECIMAL(10,2) NOT NULL 总价
status VARCHAR(20) NOT NULL 状态
created_at DATETIME NOT NULL 创建时间
updated_at DATETIME NOT NULL 更新时间

3. 数据传输对象 (DTOs)

OrderDTO

字段名 数据类型 描述
id Long 订单ID
userId Long 用户ID
homestayId Long 民宿ID
type String 订单类型
totalPrice BigDecimal 总价
status String 状态
guestName String 客人姓名
guestPhone String 客人电话
guestEmail String 客人邮箱
createdAt LocalDateTime 创建时间
updatedAt LocalDateTime 更新时间
