真实数据库表结构

1. 用户表 (user)

字段名 数据类型 约束 描述
id BIGINT PRIMARY KEY, AUTO_INCREMENT 用户ID
username VARCHAR(50) NOT NULL, UNIQUE 用户名
password VARCHAR(255) NOT NULL 密码（加密）
phone VARCHAR(20) NOT NULL, UNIQUE 手机号
role VARCHAR(20) NOT NULL, DEFAULT 'TOURIST' 角色：LANDLORD-房东，TOURIST-游客，ADMIN-管理员
avatar VARCHAR(500) DEFAULT NULL 头像URL
status TINYINT NOT NULL, DEFAULT '1' 状态：0-禁用，1-启用
create_time DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP 创建时间
update_time DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 更新时间

2. 民宿表 (homestay)

字段名 数据类型 约束 描述
id BIGINT PRIMARY KEY, AUTO_INCREMENT 民宿ID
name VARCHAR(100) NOT NULL 民宿名称
address VARCHAR(255) NOT NULL 地址
latitude DOUBLE DEFAULT NULL 纬度
longitude DOUBLE DEFAULT NULL 经度
price DECIMAL(10,2) NOT NULL 价格（每晚）
room_type VARCHAR(50) DEFAULT NULL 房型
facility VARCHAR(500) DEFAULT NULL 设施
image_url VARCHAR(500) DEFAULT NULL 封面图片地址
description TEXT DEFAULT NULL 描述
status TINYINT NOT NULL, DEFAULT '1' 状态：0-下架，1-上架，2-待审核
owner_id BIGINT NOT NULL 房东ID
create_time DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP 创建时间
update_time DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 更新时间

3. 订单表 (order)

字段名 数据类型 约束 描述
id BIGINT PRIMARY KEY, AUTO_INCREMENT 订单ID
user_id BIGINT NOT NULL 用户ID
homestay_id BIGINT NOT NULL 民宿ID
check_in_date DATE NOT NULL 入住日期
check_out_date DATE NOT NULL 退房日期
price DECIMAL(10,2) NOT NULL 订单价格
status VARCHAR(20) NOT NULL, DEFAULT 'PENDING' 订单状态：PENDING-待支付，PAID-已支付，CANCELLED-已取消，COMPLETED-已完成
pay_time DATETIME DEFAULT NULL 支付时间
create_time DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP 创建时间
update_time DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 更新时间
experience_id BIGINT DEFAULT NULL 体验项目ID
order_type VARCHAR(20) NOT NULL, DEFAULT 'HOMESTAY' 订单类型：HOMESTAY-民宿，EXPERIENCE-体验项目
experience_date DATE DEFAULT NULL 体验日期
quantity INT NOT NULL, DEFAULT 1 数量
lock_expire_time DATETIME DEFAULT NULL 库存锁定过期时间
experience_time VARCHAR(50) DEFAULT NULL 体验时间段

4. 评分表 (homestay_rating)

字段名 数据类型 约束 描述
rating_id BIGINT PRIMARY KEY, AUTO_INCREMENT 评分ID
homestay_id BIGINT NOT NULL 民宿ID
user_id BIGINT NOT NULL 用户ID
rating TINYINT NOT NULL 评分：1-5星
comment TEXT DEFAULT NULL 评论内容
create_time DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP 创建时间
update_time DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP 更新时间

5. 投诉表 (complaint)

字段名 数据类型 约束 描述
id BIGINT PRIMARY KEY, AUTO_INCREMENT 投诉ID
user_id BIGINT NOT NULL 用户ID
homestay_id BIGINT NOT NULL 民宿ID
content TEXT NOT NULL 投诉内容
status TINYINT NOT NULL, DEFAULT '0' 状态：0-待处理，1-已处理
handle_time DATETIME DEFAULT NULL 处理时间
create_time DATETIME NOT NULL, DEFAULT CURRENT_TIMESTAMP 创建时间

6. 民宿特色表 (homestay_feature)

字段名 数据类型 约束 描述
id BIGINT PRIMARY KEY, AUTO_INCREMENT 特色ID
